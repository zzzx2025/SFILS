# The App to Query, Retrieve, and Update Database

This folder contains the source code of the (minimal) application that allows user interaction with the database.

Recommend use of the app is to create a virtual enviroment. The data used should be in MySQL and MySQL is running. For details instructions on how to get the data into MySQL, create and run a virtual enviroment, and what the app does can be found in the "docs" folder in README.md
